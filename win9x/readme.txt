pccard.inf
==========

This is a sample driver that should work with Panasonic CF-VEW211 card (see ../fw/vew211* and ../fw/zv.bpd). The driver is generated using INF File Editor based on MSDN Library April 2000, https://library.thedatadungeon.com/msdn-2000-04/win95ddk/devdoc/live/win98ddk/devinst_12yd.htm and Windows 95 DDK docs/pnp.doc, https://archive.org/download/ms-win95-ddk.
